"# Ecommerce-Store" 
🛒 E-Commerce Store (Python Project)
📌 Project Overview

This project is a simple E-Commerce Store System built using Python.
It is designed to demonstrate the practical use of:

✅ if, elif, else conditions

✅ Functions

✅ Python Modules

✅ Loops

✅ Lists

✅ Basic User Interaction

The system allows users to select product categories, add items to cart, and simulate a basic shopping experience through the console.

🚀 Features

📂 Multiple Product Categories (Grocery, Electronics, Fruits & Vegetables)

🛍 Add items to cart

💰 Calculate total bill

🔄 Continuous shopping using loops

📦 Modular code structure

🧠 Clean logic using conditional statements

🛠 Technologies Used

Python 3

Conditional Statements (if, elif, else)

Functions

Custom Modules

Loops (while, for)

Lists

📁 Project Structure
ecommerce-store/
│
├── main.py              # Main program file
├── products.py          # Product categories module
├── cart.py              # Cart management module
└── README.md            # Project documentation
⚙️ How It Works

The user selects a category.

Products are displayed.

The user adds items to the cart.

The program calculates the total price.

The user can continue shopping or exit.

The project uses modular programming to separate logic into different files for better readability and scalability.
